# ODIN

ODIN is a specialized human resources chatbot for workers.

## Description

Started as an internship project, ODIN aims to make the life of users and workers easier when it comes to human resources especially the newly employed to help them start their work faster and manage their time better.

## Dependencies

ODIN uses the following libraries, software and APIs :

[Spacy](https://spacy.io/)\
[Qt6](https://www.qt.io/development/qt-framework/qt6)\
[Gemini API](https://aistudio.google.com/)\

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install the dependencies.

```bash
pip install spacy
```
```bash
pip install PyQt6
```
```bash
pip install google-generativeai
```
